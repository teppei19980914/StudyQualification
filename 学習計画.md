# 行動量係数を考慮した学習計画（Azure × セキュリティ × PM特化）

## 前提条件

- **目標職種**：クラウド・セキュリティ特化のプロジェクトマネージャ
- **主軸資格**：Azure系 / 情報処理安全確保支援士（SC） / IPAプロジェクトマネージャ（PM）
- **現在地**：2026年2月（Python 3 エンジニア認定実践試験 受験直前）
- **行動量係数**：
  - 行動量向上期：**1.4**（目安：平日2h / 休日3〜4h → 週14〜18h）
  - 通常期：**1.0**（目安：平日1.5h / 休日2.5h → 週10〜12h）
  - 行動量低下期：**0.7**（目安：平日1h / 休日2h → 週7〜9h）

| 区分 | 資格名 |

## 補足・設計方針

- **学習量より「配置」が重要**：低活動期は新規着手せず、高活動期に集中投資
- **行動量低下期は"戦略的に休む"**：無理なインプットは翌月の失速を招く
- **業務経験を積極的に活用**：Azure業務経験はAZ-104・AZ-500の理解速度を大幅に上げる
- **PM論文は日常業務から素材を蓄積**：2027年以降に備え、現業のプロジェクト記録を残しておく
- **本計画は燃え尽き防止を最優先に設計**：進捗が遅れた場合はSC-200や PM試験を1サイクル後ろへシフトしてよい
